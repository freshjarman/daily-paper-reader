## 摘要
尽管大规模预训练已彻底改变了语言建模，但其在具有结构化电子健康记录（EHR）的医疗保健领域的潜力仍未得到充分探索。我们提出了RAVEN，一种基于复发感知的下次就诊事件预测的新型生成式预训练策略，用于序列化EHR数据。利用包含超过一百万个独特个体的数据集，我们的模型学习根据患者历史自回归地生成下次就诊的令牌化临床事件。我们引入了对预测重复事件的规范化处理，并强调了基于EHR的基础模型评估中的一个关键陷阱：当新发事件与后续发生事件未被区分时，重复的事件令牌可能会夸大性能指标。此外，我们在数据受限、计算饱和的机制下实证研究了扩展行为，表明在没有相应增加数据量的情况下，单纯增加模型规模是次优的。我们通过零样本预测来评估我们的模型，以预测多种疾病的发病率；在此任务中，它与完全微调的基于表示的Transformer模型相媲美，并优于广泛使用的基于模拟的下一个令牌方法。最后，无需额外的参数更新，我们展示了RAVEN能够在有损临床代码映射和特征覆盖缺口的情况下泛化到外部患者队列。

## Abstract
While large-scale pretraining has revolutionized language modeling, its potential remains underexplored in healthcare with structured electronic health records (EHRs). We present RAVEN, a novel generative pretraining strategy for sequential EHR data based on Recurrence-Aware next-Visit EveNt prediction. Leveraging a dataset of over one million unique individuals, our model learns to autoregressively generate tokenized clinical events for the next visit conditioned on patient history. We introduce regularization on predicting repeated events and highlight a key pitfall in EHR-based foundation model evaluations: repeated event tokens can inflate performance metrics when new onsets are not distinguished from subsequent occurrences. Furthermore, we empirically investigate the scaling behaviors in a data-constrained, compute-saturated regime, showing that simply increasing model size is suboptimal without commensurate increases in data volume. We evaluate our model via zero-shot prediction for forecasting the incidence of a diverse set of diseases, where it rivals fully fine-tuned representation-based Transformer models and outperforms widely used simulation-based next-token approaches. Finally, without additional parameter updates, we show that RAVEN can generalize to an external patient cohort under lossy clinical code mappings and feature coverage gaps.

## 2. 方法论：RAVEN
*   **核心思想**：提出一种名为 **RAVEN (Recurrence-Aware next-Visit EveNt prediction)** 的生成式预训练策略。其核心是预测患者下一次就诊时可能发生的**完整临床事件集合**，而非下一个令牌。
*   **关键技术细节**：
    *   **输入表示与架构**：
        *   将患者轨迹表示为按时间排序的就诊序列，每次就诊包含一组无序的临床事件（诊断、药物、实验室结果、人口统计学等）令牌。
        *   使用基于GPT-2的仅解码器Transformer架构。
        *   引入特殊的 `<sep>` 令牌分隔就诊，并为其编码下一次就诊的时间信息（使用RoPE位置编码），使模型能根据指定未来时间进行条件预测。
        *   注意力机制在就诊内完全可见，在就诊间保持因果性，以尊重EHR数据的结构。
    *   **预训练目标 - 下次就诊多标签预测**：
        *   给定截至就诊 `v` 的历史 `H_v`，模型通过 `<sep>` 令牌的隐藏表示，经过线性投影和Sigmoid激活，预测就诊 `v+1` 时每个词汇表令牌出现的概率 `P(k|H_v, t_{v+1})`。
        *   基础损失函数为所有令牌上的二元交叉熵（BCE）。
    *   **复发感知正则化（关键创新）**：
        *   为解决模型倾向于重复预测历史中已出现的慢性病事件的问题，引入历史依赖的权重衰减机制。
        *   对于目标就诊 `v+1` 中的每个阳性令牌 `k`，根据其在历史 `H_v` 中的出现次数 `c(k, H_v)` 计算权重：`w_{v+1,k} = max(λ^{c(k, H_v)}, w_min)`。其中 `λ ∈ (0,1]` 为衰减因子（λ越小惩罚越强），`w_min`为防止权重消失的最小值。
        *   最终损失为加权二元交叉熵：对重复出现的阳性令牌给予较低的损失权重，从而鼓励模型更关注新发事件的预测信号。主实验采用全局设置 λ⋆ = 0.5。

##7优点
1.**创新的建模视角**:提出“下次就诊多标签事件”作为更适合结构化时序EHR数据的预训练目标避免了强加虚假的就诊内顺序同时支持单次前向传播完成任意未来时间的风险查询推理效率高.
2.**针对性地解决核心挑战**:提出的“复发感知正则化”机制直击慢性病建模的核心痛点有效缓解了模型的惰性记忆倾向使其更专注于学习有信息量的临床轨迹变化.
3.**全面深入的实证分析**:不仅验证了方法性能还深入探讨了正则化的影响以及不同于互联网文本数据的独特规模扩展规律为后续研究提供了重要洞见和实践指导.
4.**强调评价的真实性与公平性**:明确指出并尝试解决传统评价指标在EHR场景下的局限性倡导将新发事件的早期预警能力作为关键的评价维度提升了研究的临床相关性.

##8不足与局限
1.**数据和人群局限性**:所有主要开发和结论均基于单一大型医疗系统(NYU Langone)的数据集其编码实践人群构成和随访模式可能具有特殊性在其他系统中的具体权衡可能不同.
2.**词汇表和表示的局限性**:当前词汇表未采用分层编码系统如ICD-10因此无法从不同粒度的代码中获得通用高层信号;同时未纳入生命体征等重要特征限制了信息的完整性.
3.**真实世界应用的挑战**:模型中记录的发病时间是真实临床发病过程的滞后代理;此外尚不清楚模仿现有诊疗实践的模式是否对所有人群都是最优的.
4.**生成能力的探索有限**:目前工作主要聚焦于单步(下次就诊)预测尚未深入探索如何利用该框架进行高质量的多步未来轨迹序列生成也未探索干预性或反事实查询的能力.

*   **未来研究方向**：
    1.  **多模态与分层表示**：将生命体征、临床笔记文本、医学影像等非结构化或连续信号整合到统一的预训练框架中。探索使用ICD代码树等分层语义结构来增强模型的泛化能力和可解释性。
    2.  **因果与反事实推理**：扩展模型能力，使其不仅能预测“接下来会发生什么”，还能回答“如果采取某种干预（如处方特定药物），结果会如何改变？”这类反事实问题，这对于个性化医疗决策支持至关重要。
    3.  **长程轨迹生成与模拟**：探索基于RAVEN框架进行多步、连贯的患者未来轨迹生成，可用于创建合成数据、进行流行病学模拟或长期预后研究。
    4.  **高效微调与个性化**：研究参数高效微调技术（如LoRA、适配器），使预训练的RAVEN模型能够快速、低成本地适应新的医疗机构、特定患者亚群或新兴的临床预测任务。
    5.  **部署与验证**：在真实临床工作流中进行前瞻性验证，评估其辅助决策的实际效用、公平性和安全性，并解决模型偏差、校准和可解释性等实际部署挑战。

## 10. 总结
本文提出的RAVEN框架是构建面向电子健康记录的基础模型的一次重要且成功的探索。它通过创新的“下次就诊多标签预测”预训练目标和针对性的“复发感知正则化”，有效解决了EHR数据建模中的关键挑战——异构性、时序性以及慢性病事件的重复性。大量实验证明，RAVEN能够从大规模纵向EHR数据中学习到强大的通用表征，在零样本疾病发病预测任务上达到与需要复杂模拟或全监督微调的基线相媲美的性能，并展现出良好的跨机构泛化能力。

更重要的是，本研究超越了单纯的性能提升，深入揭示了在患者轨迹数据这一独特领域进行规模扩展的内在规律（U型曲线），并批判性地指出了传统评价指标在临床场景下的局限性。这些发现为未来EHR基础模型的研究奠定了方法论基础并指明了方向。尽管存在数据源单一、特征覆盖不全等当前局限，但RAVEN所确立的原则和展现的潜力，标志着向构建真正通用、可靠且具有临床洞察力的医疗AI助手迈出了坚实的一步。
